# aiaiaiads.com — комплект для запуска через Claude Code

Мультиязычный сайт рекламной сети на Hugo (EN / ZH / ID / RU).
Английский — основной язык, остальные подключаются на той же структуре.

## Состав комплекта

| Файл | Что это |
|---|---|
| `01-project-spec.md` | Спецификация проекта. Положите её в корень репозитория как `CLAUDE.md` — Claude Code будет читать её автоматически в каждой сессии. |
| `02-sitemap.md` | Полная карта сайта, URL-схема для всех языков, структура контента Hugo. |
| `03-design-brief.md` | Дизайн-система: цвета, типографика (с учётом китайского и кириллицы), компоненты, адаптация под языки. |
| `04-content-en.md` | Готовые английские тексты для всех страниц (оригинальные, переписанные — не копия Adsterra). |
| `05-prompts-claude-code.md` | Последовательные промты №1–7: скаффолдинг → тема → контент → i18n → переводы → SEO → проверка. |
| `06-translation-guide.md` | Глоссарий терминов для ZH / ID / RU и правила перевода, чтобы переводы были консистентными. |

## Порядок работы

1. Создайте пустую папку проекта, скопируйте туда весь комплект.
2. Переименуйте `01-project-spec.md` → `CLAUDE.md` (в корне проекта).
3. Запустите `claude` в папке проекта.
4. Выполняйте промты из `05-prompts-claude-code.md` по одному, сверху вниз.
   После каждого промта: `hugo server -D` → проверьте глазами → следующий промт.
5. Переводы (промты 5a–5c) можно запускать в любом порядке и в разных сессиях.

## Важно перед публикацией

- В текстах стоят плейсхолдеры вида `{{STAT_*}}` (объёмы трафика, число паблишеров,
  минимальные выплаты). Замените их своими реальными цифрами — выдуманные метрики
  в нише adtech быстро подрывают доверие.
- Юридические страницы (Terms, Privacy, Cookies) сгенерированы как каркас —
  их должен вычитать юрист.
- Контакты, платёжные методы и ссылки на кабинеты (login/signup) — плейсхолдеры,
  подставьте свои URL.
